## Customization

To use your own images:
1. Add your images to this folder
2. Update the `src` attributes in HTML files
3. Update the image loading functions in JavaScript files

## Image Guidelines

- **Format**: JPG or PNG
- **Optimization**: Compress images for web
- **Dark Theme**: Choose images that work with dark backgrounds
- **Neon Accents**: Images with cyan/magenta tones work best
- **Quality**: High resolution for hero images, optimized for cards

## Tools for Image Creation

- **Canva**: Create custom graphics
- **Photoshop**: Professional editing
- **GIMP**: Free alternative
- **Unsplash**: Free stock photos
- **Pexels**: Free stock photos
- **Midjourney/DALL-E**: AI-generated images

## Recommended Styles

- Dark, mysterious atmosphere
- Technology/digital themes
- Neon lighting effects
- Cyberpunk aesthetics
- Abstract patterns
- Hooded figures
- Matrix-style code
- Brain/psychology imagery
- Space/quantum themes
